# Contributing To BrowerAI Studio Labs

Version: 2026-03-20

Thank you for taking the time to review and improve the project.

## Repository Model

This repository is public and source-visible, but it is not open source. See `LICENSE.md` for the governing repository terms.

## What Is Welcome

- bug reports
- reproducible issue details
- documentation fixes
- design feedback
- security reports through the process in `SECURITY.md`

## Pull Requests

Pull requests are not automatically accepted. Code contributions should be considered proposed patches that require maintainer approval.

By submitting a pull request or patch, you confirm that:

- you have the right to submit the material
- the submission does not knowingly include unauthorized third-party code or secrets
- you understand that acceptance is at the maintainer's discretion
- you understand that accepted changes remain under this repository's licensing model unless the maintainer states otherwise in writing

## Before Submitting

- keep changes focused
- avoid including generated build output, packaged runtimes, logs, or credentials
- document any new dependency or licensing impact
- include reproduction or validation steps for behavior changes where practical

## Security Issues

Do not file public issues for security vulnerabilities. Follow `SECURITY.md`.
